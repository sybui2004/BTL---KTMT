Cần tải để chạy game Pong:

- DOSBOX: https://www.dosbox.com/download.php?main=1

Để chạy chương trình trên DOSBOX cần chạy những câu lệnh sau:

- mount c ${đường dẫn tới folder chứa file bong.asm}
- c:
- masm /a pong.asm
- ; + Enter
- link pong
- ; + Enter
- pong
